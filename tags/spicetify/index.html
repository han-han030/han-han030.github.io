Spicetify | 自訂化Spotify

什麼是Spicetify Spicetify是一個能自訂化Spotify，支援Windows, MacOS和Linux的命令列工具(Command-line tool) 如何安裝 Windows 開啟終端機(PoweShell) 輸入以下指令安裝Spicetify Spicetify CLI iwr -useb https://raw.githubusercontent.com/spicetify/spicetify-cli/master/install.ps1 | iex Spicetify Marketplace(能直接在Spotify裡安裝並管理插件) iwr -useb https://raw.githubusercontent.com/spicetify/spicetify-marketplace/main/resources/install.ps1 | iex (如果Spicetify沒有啟動的話，再繼續輸入Spicetify apply即可) MacOS 開啟終端機 輸入以下指令安裝Spicetify Spicetify CLI curl -fsSL https://raw.githubusercontent.com/spicetify/spicetify-cli/master/install.sh | sh curl -fsSL https://raw.githubusercontent.com/spicetify/spicetify-marketplace/main/resources/install.sh | sh (如果Spicetify沒有啟動的話，再繼續輸入Spicetify apply即可) 解除安裝 Windows 在終端機(PowerShell)輸入下指令 spicetify restore rmdir -r -fo $env:APPDATA\spicetify rmdir -r -fo $env:LOCALAPPDATA\spicetify MacOS spicetify restore rm -rf ~/.spicetify rm -rf ~/.config/spicetify 引用資料及相關連結 Spicetify官網 GitHub Repository
