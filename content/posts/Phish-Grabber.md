# 設定
使用`/settings`指令即可開始設定Phish Grabber

## 指令選項
當執行`/settings`指令時會顯示三個選項

* Global Scanners
伺服器的設定(非自訂)
* Server Scanners
自定的各種黑白名單
* Server sttings
建立log及忽略的頻道
:::danger
:warning: **注意**
為確保bot能夠懲處違規者，bot的身份組必須高於違規者的身份組
:::
:::info
:information_source: 在預設設定中，bot不需任何設定即可開始運作
:::
